------------------------------------------------------
	Tuned Theme
------------------------------------------------------

	Tuned Theme is a simple, clean responsive theme for Wordpress. Tuned theme looks good on any devices(mobile, tablet, nootebook, PC...). Tuned offers 10 widget boxes. Supports custom menus, cudtom header, custom background and RTL languages, translation ready.  Templates: Full Width, Front Page, Infinite Scroll Blog(with Masonry). Extras: Carousel(Slider), Hero Unit.

------------------------------------------------------
	Features
------------------------------------------------------
	
	- Responsive webdesign
	
	- Translation ready
	
	- Custom menu
	
	- Custom header
	
	-Custom background
	
	- RTL language support
	
	- Widgets: Tuned offers 10 widget boxes.
		- Main sidebar                       : This is the Main Sidebar on the right side of the page.
		- FPT - Hero Box                     : Appears on the Front Page Template. This is a box for the Hero Widget.
		- FPT - Carousel Box                 : Appears on the Front Page Template. This is a box for the Carousel Widget
		- FPT - 1 Column Grid Box            : Appears on the Front Page Template. This is a 1 Column Grid Box for the Text Widget, and for the Category Posts Widget
		- FPT - 3 Column Grid Box            : Appears on the Front Page Template. This is a 3 Column grid for the Category Posts Widget.
		- FWT - Hero Box                     : Appears on the Full Width Template. This is a box for the Hero Widget
		- FWT - Carousel Box                 : Appears on the Full Width Template. This is a box for the Carousel Widget
		- ISBT - Hero Box                    : Appears on the Infinite Scrolled Blog Template. This is a box for the Hero Widget
		- ISBT - Carousel Box                : Appears on the Infinite Scrolled Blog Template. This is a box for the Carousel Widget
		- Footer - 4 Column Grid             : Appears in the Footer Area. This is a 4 column grid.
		
	-Page Templates: Full Width, Front Page, Infinite Scroll Blog
